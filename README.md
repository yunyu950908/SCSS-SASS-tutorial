# scss/sass-tutorial

一份简易的 SCSS/SASS 指南

## 1.安装sass
```bash
gem install sass
```

## 2.监听 scss 转换为 CSS
```bash
sh ./build.sh
```

## 3.对照观察以下文件转化输出的CSS

- [basic](./input/basic.scss)

- [static_directive](./input/static_directive.scss)

- [mixin_directive](./input/mixin_directive.scss)
